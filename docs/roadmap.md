# Roadmap do Jarvis

## Objetivo Geral

Criar um assistente pessoal estilo "Jarvis", focado em automação doméstica, utilidades do dia a dia e integração com diversos dispositivos e APIs. A arquitetura ainda está aberta, incluindo a escolha da IA que será utilizada (API externa ).

---

## Etapas Principais

### 1. Planejamento

* Definir funções principais do Jarvis.
* Decidir o tipo de IA (não definido ainda).
* Escolher tecnologias principais (backend, automação, APIs, etc.).

### 2. Base do Sistema

* Criar estrutura do backend.
* Implementar comunicação entre módulos.
* Criar sistema de usuários e permissões (caso necessário).

### 3. Módulos Principais

* **Controle de dispositivos inteligentes** (luzes, tomadas, sensores, etc.).
* **Automação doméstica** (rotinas, horários, eventos automáticos).
* **Assistente conversacional** (via IA escolhida posteriormente).
* **Integração com APIs externas** (clima, notícias, calendário, etc.).

### 4. Interface

* Interface por voz.
* Interface mobile/web futuramente.

### 5. Testes e Documentação

* Documentar toda a API.
* Criar testes de integração e unitários.
* Possivelmente implementação de oculos com camera e microfones.
---

## Status Atual

O projeto está na fase de **planejamento inicial**. A IA que será utilizada ainda não foi definida.
